# Bluey Avatar Tools

Bluey Avatar Tools is a simple VRChat avatar optimization tool.

Features

• Remove PhysBones  
• Reduce textures  
• Create Quest clone  
• Save Quest prefab  

How to Install

1. Download the repository
2. Drag the Bluey folder into your Unity project
3. Open:

Tools → Bluey Avatar Tools

Works with VRChat SDK3 and Unity 2022.

# Bluey Avatar Tools

A simple VRChat avatar optimization tool designed to make creating Quest-compatible avatars easier.

Bluey Avatar Tools focuses on simplicity, safety, and accessibility.  
It includes optional voice guidance to help users who may struggle with reading instructions.

---

# Features

## Quest Avatar Optimizer
Convert PC avatars into Quest-ready versions without modifying the original avatar.

## Safe Clone System
Bluey Tools always creates a Quest clone first so the original avatar remains untouched.

## PhysBone Manager
Select which PhysBones to keep and remove the rest to improve performance.

## Material & Texture Tools
• Duplicate materials and textures safely  
• Convert shaders to `VRChat/Mobile/Toon Lit`  
• Compress textures for Quest

## Sound Remover
Automatically remove audio components that can cause Quest compatibility issues.

## Performance Scanner
Displays useful stats including:

• Triangle count  
• Renderer count  
• Material count  
• Texture count  
• PhysBone count  
• Audio sources  

## Download Size Estimator
Estimates avatar download size to help stay under the Quest 10MB target.

## Accessibility
Optional voice guidance can read instructions aloud for users with dyslexia or reading difficulties.
